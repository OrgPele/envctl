$gh-address-comments

Inspect unresolved PR review comments and review threads on the current branch PR. Address all actionable comments, update tests as needed, run `envctl test-focused --project <current-worktree-name>`, then delegate `envctl ship --project <current-worktree-name> --json` to a background subagent when available. The shipping subagent must report commit/push/PR failures, merge conflicts, failed checks, and any new actionable review comments immediately. Keep doing useful local work while CI is pending, then wait for required GitHub checks and final PR confirmation before closing out. If every comment is already resolved or non-actionable, report that evidence instead of making unnecessary edits.
