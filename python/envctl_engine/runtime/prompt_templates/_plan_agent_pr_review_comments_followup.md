$gh-address-comments

Inspect all unresolved PR review comments and review threads on the current branch PR, then address ALL actionable comments using thread-aware GitHub review evidence. Implement required fixes and update focused tests as needed. Route any follow-up handoff through `envctl ship --project <current-worktree-name> --json` when available; until then, use the compact manual fallback once: inspect status, stage intentional files, commit, push, update the PR, and wait for required GitHub checks. If every comment is already resolved or non-actionable, report that evidence instead of making unnecessary edits.
