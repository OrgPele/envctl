$gh-address-comments

Inspect unresolved PR review comments and review threads on the current branch PR. Address all actionable comments, update tests as needed, run `envctl test-focused --project <current-worktree-name>`, then run `envctl ship --project <current-worktree-name> --json`. If `ship` returns merge conflicts, resolve them from the `merge_conflicts` payload and rerun validation/ship. `ship` returns current check state without waiting, so continue useful local work while CI is pending, then wait for required GitHub checks and final PR confirmation before closing out. If every comment is already resolved or non-actionable, report that evidence instead of making unnecessary edits.
