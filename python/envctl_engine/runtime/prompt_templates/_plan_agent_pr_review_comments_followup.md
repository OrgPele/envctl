$gh-address-comments

Inspect unresolved PR review comments and review threads on the current branch PR. Address all actionable comments, update tests as needed, run `envctl test-focused`, then delegate `envctl ship --json` to a background subagent when available. The shipping subagent must report commit/push/PR failures, merge conflicts, failed checks, and any new actionable review comments immediately. Keep doing useful local work while CI is pending, and only return to the shipping lane if the subagent reports an issue or final check completion. If every comment is already resolved or non-actionable, report that evidence instead of making unnecessary edits.
