  compose process termination, and timeout recovery out of `requirements/supabase_lifecycle/compose.py`
  into `requirements/supabase_lifecycle/compose_handoff.py`.
- Keep `compose.py` as the compose command/locking/recovery coordination surface with explicit legacy
  private re-exports for tests and hidden callers that patch existing names.
- Update Supabase compose contract coverage, structure guards, and architecture inventory docs for the new
  owner boundary.

- No intended Supabase startup, compose command, network-recovery, port-publish recovery, readiness, or
  timeout behavior changed.

  polling and diagnostic logic.
